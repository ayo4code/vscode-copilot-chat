# GitHub Copilot Chat - 项目介绍

## 项目概述 (Project Overview)

GitHub Copilot Chat 是 Visual Studio Code 的一个扩展，为开发者提供基于AI的对话式编程助手功能。作为 GitHub Copilot 生态系统的重要组成部分，这个扩展与核心的 GitHub Copilot 扩展协同工作，提供更智能、更直观的编程体验。

## 核心功能 (Key Features)

### 🤖 智能对话助手
- **聊天界面**: 提供对话式AI助手，支持自然语言交互
- **上下文感知**: 理解当前代码上下文，提供相关建议
- **多轮对话**: 支持连续的对话交互，保持上下文连贯性

### ✏️ 内联编辑功能
- **Ctrl+I 内联聊天**: 直接在编辑器中进行AI辅助编辑
- **实时代码建议**: 在编写代码时提供智能提示和补全
- **代码重构**: AI辅助的代码优化和重构建议

### 🎯 智能代理模式
- **自主编程任务**: AI代理可以执行多步骤编程任务
- **错误自动修复**: 自动处理编译和lint错误
- **测试监控**: 监控终端和测试输出，持续优化

### 🔍 智能搜索与发现
- **工作区搜索**: 大型代码库的智能分块搜索
- **语义搜索**: 基于语义理解的代码搜索
- **相关文件发现**: 智能发现相关的代码文件和上下文

### 🛠️ 开发工具集成
- **TypeScript 支持**: 深度集成TypeScript语言服务
- **测试生成**: AI驱动的测试用例生成
- **代码审查**: 智能代码审查和改进建议
- **调试支持**: 智能调试建议和问题诊断

## 技术架构 (Technical Architecture)

### 🏗️ 项目结构
```
src/
├── extension/          # 扩展特定功能
│   ├── chat/          # 聊天功能
│   ├── inlineChat/    # 内联聊天
│   ├── inlineEdits/   # 内联编辑
│   ├── conversation/  # 对话管理
│   ├── context/       # 上下文解析
│   ├── prompts/       # 提示工程
│   └── tools/         # AI工具集成
├── platform/          # 平台服务
│   ├── openai/        # OpenAI API集成
│   ├── embedding/     # 向量嵌入
│   ├── search/        # 搜索服务
│   └── workspace/     # 工作区管理
└── util/              # 通用工具
    ├── common/        # 共享工具
    └── vs/           # VS Code 相关工具
```

### 💻 技术栈
- **主要语言**: TypeScript
- **UI框架**: React + TSX (使用 @vscode/prompt-tsx)
- **运行时**: Node.js (>=22.14.0)
- **构建工具**: ESBuild
- **测试框架**: Vitest + Mocha
- **代码规范**: ESLint + Prettier

### 🔌 VS Code集成
- 深度集成多个VS Code提议API
- 支持聊天参与者(Chat Participants)
- 语言模型系统集成
- 内联补全增强功能
- AI文本搜索提供者

## 核心组件 (Core Components)

### 聊天系统 (Chat System)
- **默认代理**: 主要的对话AI助手  
- **设置代理**: 处理Copilot初始设置和引导
- **工作区代理**: 专门用于工作区级别操作
- **代理模式**: 自主多步骤任务执行

### 请求处理流程 (Request Processing)
1. **输入解析**: 解析用户输入的参与者、变量、斜杠命令
2. **上下文解析**: 收集相关代码上下文、诊断信息、工作区信息
3. **提示构建**: 使用上下文和意图检测构建提示
4. **模型交互**: 向适当的语言模型发送请求
5. **响应处理**: 解析和解释AI响应
6. **动作执行**: 应用代码编辑、显示结果、处理后续操作

### 语言模型集成 (Language Model Integration)
- 支持多个提供商 (OpenAI, Anthropic等)
- 模型选择和切换功能
- 配额管理和回退处理
- 自定义指令集成

## 开发指南 (Development Guidelines)

### 环境要求 (Requirements)
- Node.js >= 22.14.0
- npm >= 9.0.0  
- Python >= 3.10, <= 3.12
- VS Code ^1.102.0-20250627

### 快速开始 (Quick Start)
```bash
# 安装依赖
npm install

# 获取token
npm run get_token

# 构建项目 (Cmd+Shift+B)
# 或使用 "Launch Copilot Extension - Watch Mode" 调试配置
```

### 代码规范 (Coding Standards)
- **缩进**: 使用制表符(tabs)，不使用空格
- **命名约定**: 
  - `PascalCase` 用于类型和枚举值
  - `camelCase` 用于函数、方法、属性和局部变量
- **字符串**: 
  - "双引号" 用于需要本地化的用户可见字符串
  - '单引号' 用于内部字符串
- **函数**: 优先使用箭头函数 `=>` 而不是匿名函数表达式

### 测试体系 (Testing Framework)
- **单元测试**: Vitest 用于隔离组件测试
- **集成测试**: VS Code扩展主机测试用于API集成
- **仿真测试**: 端到端场景测试 (`.stest.ts` 文件)
- **性能测试**: 成本高昂的LLM交互测试，包含缓存机制
